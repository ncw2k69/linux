## APT
Open
```
sudo nano /etc/apt/apt.conf
```
Add
```
APT::Install-Recommends "false";
APT::Install-Suggests "false";
```
## Install Apps
```
sudo apt install htop btop iotop ncdu mc net-tools curl ssh sshfs lm-sensors lshw sudo
```
## Sudo
Add user in list
```
nano /etc/sudoers.d/me
```
Content
```
your-user-name-here     ALL=(ALL)       ALL
```
## Fixing HALT on Power Off or Reboot
There is an issue related to the HSUART DMA support in the kernel that will hang on reboot or shutdown. To prevent this, add those lines to `/etc/modprobe.d/blacklist.conf`
```
blacklist dw_dmac_core
install dw_dmac /bin/true
install dw_dmac_core /bin/true
```
Then, update `initramfs`
```
sudo update-initramfs -u
```
Fix Source: [Debian Wiki](https://wiki.debian.org/InstallingDebianOn/Dell/Wyse%203040#System_Summary)
